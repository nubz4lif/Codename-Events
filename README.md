# Nubz Codename Events and Notes

This is a repo containing [Codename Engine](https://github.com/CodenameCrew/CodenameEngine) events and notes that I have made

These are all free to use and under the MIT license (though, some credit would be nice :3)

* Standalone events go in the "events" folder (pairs of .json and .hx)
* Standalone notes go in the "notes" folder
* Events/notes that require additional assets go in the "packs" folder
